Tech Specs

Hardware​

ComponentDetails
MCUESP8266 (Tensilica L106, 80 / 160 MHz)
Wi-Fi802.11 b/g/n, 2.4 GHz, AP+STA simultaneous
RF modulenRF24L01+ with PA and LNA amplifier, SMA antenna connector
RF frequency2.400 – 2.525 GHz, 125 channels, 1 or 2 Mbps
DisplaySH1106 128×64 OLED, I²C
Input6 tactile buttons via ADC voltage divider
StorageEEPROM — 5-slot credential vault
USBUSB-C (power and firmware flash)
PCBCustom RDH ROBOTICS board

Pinout​

nRF24L01
  CE   →  GPIO16
  CSN  →  GPIO15
  MOSI →  GPIO13
  MISO →  GPIO12
  SCK  →  GPIO14

SH1106 OLED
  SDA  →  GPIO4
  SCL  →  GPIO5

ADC buttons (all on A0 via voltage divider)
  UP     →  135
  DOWN   →  579
  LEFT   →  426
  RIGHT  →  634
  ENTER  →  729
  BACK   →  687
