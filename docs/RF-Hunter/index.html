RF-Hunter

A pocket-sized wireless security research device built by RDH Robotics.

Authorized use only

This device is for research, education, and authorized security testing only. Using it against networks or devices you don't own or have written permission to test is illegal.

What is it?​

The RF-Hunter is a standalone device — no laptop, no app, no setup required. You control everything from the OLED display and six buttons on the front.

It runs three tools:

ToolWhat it does
RF-HunterStudies RF interference across the 2.4 GHz band using an nRF24L01 radio module
Deauther SuiteScans nearby Wi-Fi networks and studies 802.11 management frame behavior
WiPhiSets up a test captive portal to study how credential phishing works

All three launch from the same menu. Switch between them without rebooting.

Who is this for?​

  • Students learning wireless security concepts hands-on
  • Hobbyists exploring how 2.4 GHz radio works
  • Security researchers running authorized penetration tests
  • CTF participants who need a portable RF tool

You don't need to know how to code to use it. If you want to modify the firmware or add new modes, the full source is on GitHub.

  • RF-Hunter assembled and tested unit
